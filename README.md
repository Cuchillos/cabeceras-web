# cabeceras-web
 Obtener las cabeceras de una web


Como usarlo:
-------------
Termux:
```
git clone https://github.com/Cuchillos/cabeceras-web
cd cabeceras-web
pip install -r requirements.txt
python cabeceras-web3.py -h
```

PC:
```
pip install -r requirements.txt
python cabeceras-web3.py -h
```
