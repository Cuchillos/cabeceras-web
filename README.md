# cabeceras-web
 Obtener las cabeceras de una web


Como usarlo:
-------------
Termux:
```
git clone https://github.com/Cuchillos/cabeceras-web.git
cd cabeceras-web3.py
pip install -r requirements.txt
python cabeceras-web.py
```

PC:
```
pip install -r requirements.txt
python cabeceras-web.py
```
